## 📌 Executive Overview (نبذة تنظيمية عن المختبر)
**Tree Algorithms Lab** is an interactive browser-based laboratory designed for computer science students and engineers to construct, manipulate, balance, and visualize non-linear hierarchical tree structures and graph algorithms in real time.

Unlike HTML5 Canvas implementations that obscure node interaction behind coordinate math, this studio leverages **Pure SVG DOM Elements**. Every rendered node and edge is a genuine DOM element, providing seamless tap-to-add, click-to-delete, and responsive drag ergonomics across desktop and touch screen viewports.

## ⭐ Visualized Data Structures (الهياكل الشجرية المُنفَّذة)

### 1. 🌲 Binary Trees & Binary Search Trees (BST)
* **Interactive Tree Builder:** Level-by-level BFS level ordering or recursive BST ordering.
* **Logarithmic Pruning Visualizer:** Real-time search step animation highlighting discarded subtrees.

### 2. ⚖️ AVL Self-Balancing Trees
* **Strict Height Invariant:** Enforces Adelson-Velsky Landis invariant $|BF| \le 1$.
* **Step-by-Step Balancing Animation:** Visual breakdown of single (LL, RR) and double (LR, RL) rotations.

### 3. ⭐ Splay Self-Adjusting Trees (Tab 08)
* **Temporal Locality Simulation:** Moves recently accessed elements to the root via splaying mechanics (Zig, Zig-Zig, Zig-Zag).

### 4. 🔴 Red-Black Trees (Tab 09)
* **5 Core Invariants Mechanics:** Demonstrates deterministic red/black node coloring, root black invariants, and equal black-height path verifications.

### 5. 🧮 Algebraic Expression & m-ary Trees
* **Numeric Expression Tree:** Postfix AST execution engine for math equations.
* **Symbolic Expression Tree:** Automatic tree calculus differentiation rules.
* **m-ary Conversion:** Knuth's Left-Child Right-Sibling (LCRS) isomorphism mapping.
